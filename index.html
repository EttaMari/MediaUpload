Media Upload

Getting Started

This article is meant to take you through the steps to implimenting a functioning html script that will load pictures into a database. Before we begin you need to set up your table in your database. We used a slim-skeleton on a vagrant program. If you do not have this set up we suggest going here and do so. Also you should have mysql downloaded if you dont navigate to where your vagrant file and use the commands below to install mysql.

sudo apt-get install -y mysql-server mysql-server libapache2-mod-auth-mysql php5-mysql

Now that you have all those things we are ready to begin!

Creating The Table

Once you've logged into mysql you need to create that database and table you will be storing the image to:

  • Create a database, ours is named databaseimage: CREATE database databaseimage;
  • Switch over to your database: use databaseimage;
  • Create a table, ours is named store: CREATE table store /n

Creating The Attributes

Once you've created the table store, you also need to create the attributes that are under the table since table can be created with at least one column.

  • Generate an opening parenthesis right after creating the table: (
  • Create the first attribute inside the parenthesis, our first attribute is id with a type int, two constraints NOT NULL and PRIMARY KEY and one field AUTO INCREMENT: 'id' INT NOT NULL AUTO INCREMENT PRIMARY KEY, //
  • Create a table, ours is named store: CREATE table store /n
